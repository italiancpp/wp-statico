++it promuove la pubblicazione di articoli, rubriche,lezioni…

Fin da pochi mesi dopo la nostra fondazione (2013), inostri articoli hanno supportato un nuovo modo di fare blog sul C++:snippet compilabili, ovvero la possibilit di immergere direttamente nell’articolo degli snippet di codice C++ modificabili ed eseguibili da un vero compilatore (e.g. clang):

#include <iostream>
using namespace std;

int main()
{
   cout << "I nostri articoli!\n";
}

Vuoi scrivere articoli su questo sito? Le regole sono semplici:
    1. Diventa un autore
    2. Attenendoti ad alcune linee guida, crea il tuo articolo e invialo per la revisione
    3. Alcuni revisori verificheranno laconformited icontenutidella tua pubblicazione
    4. Se approvato, il tuo articolo verrpubblicato e pubblicizzato!
